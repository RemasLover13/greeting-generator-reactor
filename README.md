# 🎉 Генератор поздравлений с AI

Современное веб-приложение для создания уникальных поздравлений с использованием искусственного интеллекта. Генерирует персонализированные тексты и изображения в различных стилях для любых праздников.

## ✨ Особенности

### 🤖 AI-генерация текста
- Уникальные поздравления с учетом всех параметров
- Поддержка 6 языков
- 6 стилей тона (от официального до 18+)

### 🎨 AI-генерация изображений
- **8 художественных стилей**: Реалистичный, Аниме, Мультяшный, Акварель, Масляная живопись, Цифровой арт, Пиксельная графика, Минимализм
- **Без текста на изображениях** — только чистая визуальная эстетика
- Высокое качество 4K
- Квадратный формат (1024x1024)

## 🛠️ Технологии

### Frontend
- **React 18** — библиотека для построения интерфейсов
- **TypeScript** — типобезопасность и лучшее DX
- **Tailwind CSS** — стилизация и адаптивность
- **Vite** — быстрая сборка и разработка

### AI Сервисы
- **OpenRouter API** — генерация текста (модель Google Gemini 2.5 Flash Lite)
- **Pollinations API** — генерация изображений (модель FLUX)

### TypeScript особенности
- Enum для ограниченных наборов значений (праздники, тона, стили)
- Union types для языков
- Generic типы для Promise и API ответов
- Type Guards для безопасной обработки ошибок
